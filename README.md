# Medical Insurance Cost Predictor 💉💰

This project predicts medical insurance costs based on personal and lifestyle information using machine learning.

## 📌 Project Overview
Medical insurance costs depend on several factors such as age, BMI, smoking habits, and region.  
This application helps estimate medical costs by analyzing these factors.

## 📊 Dataset Description
The dataset contains 10,000 records collected between 2010 and 2020.

### Features:
- Age (18–65)
- Sex (Male / Female)
- BMI (15–40)
- Number of Children (0–5)
- Smoker (Yes / No)
- Region (Northeast, Northwest, Southeast, Southwest)

### Target:
- Medical Insurance Cost

## 🧠 Machine Learning Model
- Decision Tree Regressor
- Trained on real-world medical insurance data

## 🖥️ Web Application
- Built using Streamlit
- User-friendly interface
- Input validation included
- Displays predicted medical cost in INR

## 🚀 How to Run the Project
1. Install required libraries
2. Run the Streamlit app
3. Enter patient details
4. Get predicted medical cost

## 📌 Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
